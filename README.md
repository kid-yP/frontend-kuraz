# Task Manager App

A simple task manager built with **React** and **Tailwind CSS**. Users can **add**, **delete**, **mark tasks as completed**, and **filter** tasks.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks: **All**, **Pending**, **Completed**
- Styled using **Tailwind CSS**
